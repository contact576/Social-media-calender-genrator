How to use this tool

  1. On Intake form, fill the client's details. Required fields are marked *.
  2. Press Generate prompts. The Prompts tab fills with S1–S8.
  3. Open a Claude chat that has the Apify + Higgsfield + Google Drive connectors.
  4. Copy S1, paste, let it run, and let it save to the Drive vault. Then S2, S3 … through S8. Do not skip ahead — each step loads the previous step's saved file.
  5. Keep the whole S3→S4 decode in one session — Instagram video links expire within hours.

If something goes wrong

  • "Approve permissions" error from an actor? Copy the link it shows to the owner to approve once in the Apify console, then re-run that prompt.
  • A video link gives a 403 / expired? Re-scrape that single reel in the same chat before decoding it. Never let the model write a transcript from the caption.
  • A later prompt says a vault file is missing? Go back and make sure the earlier step actually saved to Drive (each step echoes its saved path).
  • Reconciler banner is red at the top? The templates and form fields are out of sync — tell the developer; do not run the prompts.

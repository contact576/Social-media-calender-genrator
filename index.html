Before your first run — one-time setup, about 15 minutes

  1. A paid Claude plan (claude.ai). The instructions this tool writes are run inside Claude — it does the research and writing.
  2. An Apify account with a payment method (apify.com). Apify reads the public Instagram data. It's pay-as-you-go: a full run typically costs $2–6, and the prompts meter themselves so a run can't overspend.
  3. A Google account. Every step saves its results to a Google Drive folder automatically — that folder is your project file.
  4. In Claude: Settings → Connectors → connect Apify and Google Drive, then open a new chat. That chat is your "connected chat" — every prompt from this tool gets pasted there. (Using Claude Code mode instead? Connect the same two connectors in Claude Code.)

What you get at the end: a research report on the reels actually going viral in your niche (with the receipts), a 30-day posting calendar, a ready-to-shoot two-column script for every slot, and a branded printable report — typically within one afternoon.

How to use this tool

  1. On Intake form, fill the client's details. Required fields are marked *. Not sure what good answers look like? Press See a filled-in example.
  2. Press Generate prompts. The Prompts tab fills with Steps 1–5.
  3. Copy Step 1, paste it into your connected Claude chat, let it run — it ends by printing the Drive file it saved. Then Step 2 … through Step 5. Do not skip ahead — each step loads the previous step's saved file.
  4. Run Steps 2 and 3 back-to-back in one sitting — Step 2 collects Instagram video links that expire within hours; Step 3 needs them live.
  5. Each step card shows a "Done when" line — check it before moving on. (Heads-up: the saved file names don't match the step numbers — Step 2 saves s2 + s3, Step 5 saves s8. That's normal.)
  6. When Step 5 prints the final report, paste it into the Beautifier tab → Render → Print/Save as PDF.

If something goes wrong

  • The chat says it needs permission or to "rent an actor"? It will show a link. Open it, approve or rent it in your own Apify account (you're the owner), then paste the same prompt again. The only rental you might ever need is the backup transcriber (~$5/mo) — the default one is pay-per-use pennies.
  • A video link gives a 403 / "expired"? Tell the chat: "re-scrape that one reel and continue". Never accept a transcript it wrote from the caption — the prompts forbid it, and so should you.
  • A later prompt says a vault file is missing? The step that writes it is named in the message. Re-run that step and confirm it prints the saved Drive path this time.
  • The research came back skewed to one country? The prompts run extra geo-tagged searches for your market and report an ORIGIN MIX line. If your local market is genuinely thin on Instagram, the report says so honestly instead of pretending.
  • Red banner at the top of this page? The templates and form fields are out of sync — don't run the prompts; tell the person who maintains this tool.
  • Still stuck? Email the person who set you up with this tool — include a screenshot of the step and the chat's last reply.
